自定义自己的域名

觉得自己的域名不够酷？想要自定义自己的域名？可以试试自定义域名

  1. 首先需要注册一个域名，可以在GoDaddy上购买一个域名，这里我选择taoxiyang.me，购买的过程省略
  2. 将自己的域名注册到一个DNS服务提供商，在这里我选择阿里云域名解析服务
  3. 在github page设置自己的Custom domain
  4. 访问taoxiyang.me

可以看到自定义的域名生效了

Custom domain重置问题

每次部署到git，会发现GitHub Pages的Custom domain选项会被重置，解决这个问题，需要在博客目录的source/目录下新建一个CNAME文件，将自己的Custom domain写进文件

这样，每次部署到git，就不会把Custom domain重置了
